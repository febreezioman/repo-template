# Papers

Master list of relevant literature for this project.

## How to add a paper

1. Create a folder: `papers/<AuthorYear_keyword>/` (e.g. `Smith2023_ATAC/`)
2. Drop the PDF and any supplementals in that folder.
3. Add a `notes.md` with your reading notes (see template below).
4. Add a row to the table below.

> PDFs are excluded from git by default (see `.gitignore`).  
> To version-track PDFs, set up [git-lfs](https://git-lfs.com): `git lfs track "papers/**/*.pdf"`
